Getting Started

Make sure you are using a virtual environment of some sort (e.g. virtualenv or pyenv).

To simply run the app

pip install -r requirements.txt
./manage.py migrate
./manage.py loaddata sites
./manage.py runserver

For developers

pip install -r requirements_dev.txt
